<p><h1># coordinates movement function moveByCoordinates(x,y,z)</h1></p>

<p>moveByAbsoluteCoordinates(x,y,z,pitch,roll) 
- Moves by absolute coordinates, ex. with robot in position = (0,0,0)
 a move by absolute coordinates (2,2,2) would result in robot position = (2,2,2)</p>

<p>moveByRelativeCoordinates(x,y,z,pitch,roll)</p>
- Moves by relative coordinates, ex. with robot in position = (2,2,2)
 a move by relative coordinates (2,2,2) would result in robot position = (4,4,4)</p>

<p><h1> gripper functions </h1></p> 

<p> OpenGripper() - opens the gripper</p>
<p>CloseGripper() - closes the gripper around a subject</p>

<p><h1>standard robot functions</h1></p>

<p>isOnline() - returns true if robot is connectable</p>
<p>Initialization() - Initializes the robot with standard parameters</p>
<p>seekHome() - makes the robot go to home position</p>
<p>stopAllMovement() - stops movement for all axes moving</p>

<p><h1>axis movement functions</h1></p>

<p>moveBase(speed) - moes base axis with parameter of speed in percentage (0-100%)</p>
<p>moveShoulder(speed) - moves shoulder axis with parameter of speed in percentage (0-100%)</p>
<p>moveElbow(speed) - moves elbow axis with parameter of speed in percentage (0-100%)</p>
<p>moveWristRoll(speed) - moves wristroll axis with parameter of speed in percentage (0-100%)</p>
<p>moveGripper(speed) - moves gripper axis with parameter of speed in percentage (0-100%)</p>
<p>moveConveyorBelt(speed) - moves conveyor belt with parameter of speed in percentage (0-100%)</p>
