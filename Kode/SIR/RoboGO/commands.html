Coordinates movement function moveByCoordinates(x,y,z)

moveByAbsoluteCoordinates(x,y,z,pitch,roll)

moves by absolute coordinates, ex. with robot in position = (0,0,0) a move by absolute coordinates (2,2,2) would result in robot position = (2,2,2)

moveByRelativeCoordinates(x,y,z,pitch,roll)

moves by relative coordinates, ex. with robot in position = (2,2,2) a move by relative coordinates (2,2,2) would result in robot position = (4,4,4)

Gripper functions

OpenGripper()

opens the gripper

CloseGripper()

closes the gripper around a subject

Standard robot functions

isOnline()

returns true if robot is connectable

Initialization()

Initializes the robot with standard parameters

seekHome()

makes the robot go to home position

stopAllMovement()

stops movement for all axes moving

Axis movement functions

moveBase(speed)

moves base axis with parameter of speed in percentage (0-100%)

moveShoulder(speed)

moves shoulder axis with parameter of speed in percentage (0-100%)

moveElbow(speed)

moves elbow axis with parameter of speed in percentage (0-100%)

moveWristRoll(speed)

moves wristroll axis with parameter of speed in percentage (0-100%)

moveGripper(speed)

moves gripper axis with parameter of speed in percentage (0-100%)

moveConveyorBelt(speed)

moves conveyor belt with parameter of speed in percentage (0-100%)

insertBox(positionid,length,width,depth,weight)

inserts box into database with given position ID, length, width, depth, weight

removeBox(boxid)

removes box from database with given Box ID
